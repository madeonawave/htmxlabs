## install
 - git clone the project
 - $ uv sync
 - create a file site/config/.env   
   * SECRET_KEY=secret-key-here 
   * DEBUG=True 
   * ALLOWED_HOST=*,htmxlabs.com,localhost
   * CSRF_TRUSTED_ORIGINS=http://localhost
